Sales Analytics Dashboard (Power BI + SQL) (*In Progress)

1.	 Project Overview
   
This project focuses on building an end-to-end  Sales & Revenue Intelligence Dashboard using SQL and Power BI. It transforms raw retail sales data into actionable business insights, enabling stakeholders to track performance across products, customers, regions, and time periods.


   
2.   Business Problem
   
⦁	Organizations often manage sales data across multiple systems without consolidated reporting.

⦁	 This leads to Limited revenue visibility

⦁	 Delayed decision-making

⦁	 Inability to track growth trends.


3.   This dashboard solves these challenges by enabling  real-time sales monitoring and KPI tracking .

   

4.   Tech Stack & Tools


   
 ⦁	Power BI (Data modeling, DAX, visualization)

 ⦁	SQL (MySQL / SQL Server) — data extraction & cleaning
 
 ⦁	Power Query
 
 ⦁	Excel/CSV raw dataset
 
 ⦁	Star Schema Data Modeling
 


5.    Key Objectives
  
  
⦁	 Build a centralized sales reporting system  

⦁	 Track revenue, profit, customer growth, and category performance

⦁	 Implement reusable data model and DAX measures  

⦁	 Enable drill-through and interactive exploration  



6.    Data Preparation Steps
  
  
⦁	 Removed duplicates and nulls

⦁	 Standardized date formats

⦁	 Normalized product and category data

⦁	 Created  FactSales and  Dimension tables

⦁	 Added Date table using `CALENDAR()` DAX
